Top Author list By Count Post

== Description ==

Display top authors list by post.

Plugin Features

* Three Diffrent Style.
* Add Anywhere via widgets.
* Post Count.
* Unlimited Authors List. 


Live Preview: http://kentothemes.com/demo/kento-top-author/

== Installation ==

Via WordPress -

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate please go widget page you will see widget "Kento: Top Authors" 
4. Drag this to your sidebar, Set value for settings<br />
	<strong>Widget Title:</strong> <br />
	<strong>How Many Author to show:</strong><br />
	<strong>select style radio button</strong><br />
5. Hit save
